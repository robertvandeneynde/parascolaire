{{Option 3 : Installer via pipwin|Option 3: Install via pipwin}}

Avant, pygame était difficilement installable via pip, et donc on devait utiliser pipwin ou les whl... Maintenant c'est facile !

Windows (via pip)

  • Téléchargez python version 3.9 sur python.org.
  • Pour télécharger pygame nous allons utiliser pip.
  • Ouvrez cmd (cliquez sur le bouton windows en bas à gauche de l'écran → tappez cmd).
  • Entrez la commande suivante : 
    
                py -3 -m pip install pygame

    Elle prend un peu de temps, genre quelques secondes :D

Si jamais dans le futur vous voyez un tutoriel qui vous dit de tapper une commande du style python TRUC, chez vous ça sera py -3 TRUC ou py -2 TRUC si vous devez utiliser python version 2 ou py -3.7 TRUC si vous devez utiliser une version explicite de python, comme 3.7.

Vous pouvez maintenant commencer à coder en choisissant un IDE !

Windows (via pipwin)

  • Téléchargez python version 3.9 sur python.org.
  • Pour télécharger pygame nous allons utiliser pipwin.
  • Ouvrez cmd (cliquez sur le bouton windows en bas à gauche de l'écran → tappez cmd).
  • Entrez la commande suivante : 
    
                py -3 -m pip install pipwin
  • Entrez la commande suivante : 
    
                py -3 -m pipwin install pygame

    Elle prend un peu de temps, je n'y peux rien ^^

Si jamais dans le futur vous voyez un tutoriel qui vous dit de tapper une commande du style python TRUC, chez vous ça sera py -3 TRUC ou py -2 TRUC si vous devez utiliser python version 2 ou py -3.7 TRUC si vous devez utiliser une version explicite de python, comme 3.7.

Si on vous dit d'installer un paquet via pip, par exemple pip install pillow ça sera py -3 -m pipwin install pillow chez vous.

Si la commande précédente ne marche pas, c'est probablement que le package n'est pas disponible via pipwin, regardez l'Option 4.

Vous pouvez maintenant commencer à coder en choisissant un IDE !

{{Une ancienne version de python existe : python 2, pour savoir les différences avec python 3, lisez la remarque ci-dessous. | A former python version exist: python 2, to know the differences between python 2 and python 3, read the note below. }}

Windows

  • Déterminez si votre version de python est 32 bits ou 64 bits, quand vous allez dans le menu windows et tappez python cela devrait être marqué. Même si votre ordinateur est 64 bits, il se peut et c'est très probable que vous ayez téléchargé python 32 bits (et ça marchera très bien). Pour savoir si votre ordi est 32 ou 64 bits, allez dans Panneau de configuration → Système.
  • Téléchargez pygame ici, et téléchargez le fichier whl en fonction de votre version de python (par exemple, j'ai pris pygame‑1.9.3‑cp37‑cp37m‑win32.whl car j'ai python 32 bits version 3.7).
  • Ce fichier doit être dans votre dossier Téléchargements, il est probable qu'il y est déjà.
  • Ouvrez cmd (cliquez sur le bouton windows en bas à gauche de l'écran → tappez cmd).
  • Entrez la commande suivante : 
    
                py -3 -m pip install Downloads\pygame-1.9.3-cp37-cp37m-win32.whl

Si jamais dans le futur vous voyez un tutoriel qui vous dit de tapper une commande du style python TRUC, chez vous ça sera py -3 TRUC ou py -2 TRUC si vous devez utiliser python version 2 ou py -3.7 TRUC si vous devez utiliser une version explicite de python, comme 3.7.

Si on vous dit d'installer un paquet via pip, par exemple pip install pillow téléchargez le whl sur le site non officiel (pour trouver vite un paquet, utilisez Ctrl+F) et exécutez la commande qu'on vient de faire avec le bon nom de fichier.

Vous pouvez maintenant commencer à coder en choisissant un IDE !

Mac

  • Téléchargez la dernière version de python (3.7) sur python.org.
  • Installez XCode via l'AppStore... Ça risque de prendre un peu de temps...
  • Ouvrez un terminal (cliquez sur l'icône de recherche en haut à droite de l'écran et tappez Terminal ou bien suivez ceci) et entrez les commandes suivantes :
    • Cette commande ruby qui installe homebrew (brew) : 
      
                        /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
    • Ces commandes brew qui installent des trucs nécessaires (xquartz, sdl, des bibliothèques d'images et de son, et mercurial): 
      
                        brew install caskroom/cask/brew-cask
                        brew cask install xquartz
                        brew install sdl sdl_image sdl_ttf portmidi libogg libvorbis
                        brew install sdl_mixer --with-libvorbis
                        brew tap homebrew/headonly
                        brew install smpeg
                        brew install mercurial
    • Cette commande pip installe pygame en une seule ligne : 
      
                        pip3 install hg+http://bitbucket.org/pygame/pygame
